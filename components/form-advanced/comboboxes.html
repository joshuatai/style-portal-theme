The icon class should only be used on elements that do not contain any text or child elements. We recommend wrapping any or element.

A combo box allows the user to type in a single-line input to limit the search, and select an item from a list of matched options. We recommend using a combo box when you have a list that contains more than 13 items. In case when the list contains less than 13 items, use a drop-down menu.
