Resizable handler

Table columns may be resized by dragging border on table headers.

  • When a resizable columns border is hovered over on table headers, display a col-resize cursor and show the border in another color to indicate that the column is resizable, and then adding a translucent border when users drag the boundary.
  • Use a minimum width of 56px for resizable columns if that columns without critical limitation of content.
  • Recommend to support double-click the right border of a table header cell to increase the column width to accommodate the largest entry in the column.
