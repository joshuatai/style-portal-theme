A toolbar contains actions that impact the whole table. Please be aware that actions should corresponding multi-selection items, do not block the actions work when selected more than one item.

  • The toolbar always appears above the table, the background of toolbar based on types of table or different situation.
  • In case the user has to use the table action very often, consider to freezes it above the table while scrolling down the page.
  • Use the primary button on tool bar cautiously, use it only to highlight the most important action in the table.
  • You can put control into the toolbar for different usage. For the specific actions of control, you can display controls directly; For the generic actions of control, you may display controls after users selected items.
