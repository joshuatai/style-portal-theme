Modal dialogs are used to get a response from a user before other features can be assessed from the main page. Use title case for the titles.

There are four common modal sizes. You may also adjust the size as required.

  • Extra small:W400 x H240 px (minimum height)
  • Small:W544 x H304 px (minimum height)
  • Medium:W688 x H304 px (minimum height)
  • Large:W928 x H304 px (minimum height)
