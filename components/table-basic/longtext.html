About the infotip style, please refer to Tooltip component.

There are two options to handle text that does not fit into a table cell.

  • Non-truncated: Display all by warp the text with extend height of rows. Table header should be middle vertical alignment, table content should be top vertical alignment.
  • Truncated: Truncate text (the ending or in the middle) with ellipsis and adding a infotip to display it in full on hover. A infotip should take effect when the user hover anywhere in a table cell which is truncated.
