Search container

Use a element to create a container that wraps the input and the button. For example, .input-group is used as a search container in Bootstrap.

Search button

Click the search button to start querying for the typed text.

  • Use the default button style for the search button. You may also use the primary or emphasis style for emphasis.
  • You can include a reset button for clearing up the search text. Only show the reset button when text is typed into the search box and the search is complete.
