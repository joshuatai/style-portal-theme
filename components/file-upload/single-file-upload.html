File upload container

Create the container class, and wrap the input file, file upload buttong, and file information container with a element.
Input file

Use the element with certainly id attribute.
File upload button

Use the element with for attribute which refer to ID. This should apply Button style, please refer to Buttons.

File information container

Use block-level elements on information containers which should wrap the file name, file size, and the cancel icon.
File size
Icon

Icon classes should be only used on elements that contain no text content and have no child elements. We recommand to use element.
