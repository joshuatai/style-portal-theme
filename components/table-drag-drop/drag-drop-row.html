Tables with draggable rows allow users to rearrange the order of records within tables.

  • Only allow one row to be dragged at a time.
  • Display a visual hint (a hand cursor) while hovering over a draggable row.
  • Display a visual hint (a move cursor) while dragging a row.
  • Display a visual hint (gray background) to indicate where a dragged row will be dropped.
  • Display a visual hint (a not-allowed cursor) if a row is dragged somewhere it cannot be dropped.
  • Return a row to its original position in the table if the row is dropped where it cannot be dropped.
  • Drag-and-drop tables should have less than 100 rows and not display pagination controls.
