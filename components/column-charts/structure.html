Column charts display vertical bars that are used to compare multiple values over time or over values containing an intrinsic order. The best way to represent these values is to use the horizontal axis.

  • Use the qualitative palette color to display category items.
  • Ensure the category has an intrinsic order. If not, consider useing a bar chart instead.
  • Avoid the labels are displayed at 45° or 90° on X-axis, it makes labels difficult to read.
  • Shorten well-known names. Long data labels affect the readability, when labels are well-known terms, it is best to shorten them. (example: Jan, Feb, Mar...)
  • Avoid using gradient color and 3-D bar charts.
