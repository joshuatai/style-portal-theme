Bar charts are used to compare data across categories. They are especially effective when you have numerical data that splits nicely into different categories. Examples: Volume of scanned traffic in different continents.

  • Use the qualitative palette color to group items into categories. Example, use blue for Ransomware, orange for web reputation.
  • Ensure the numerical axis starting at zero.
  • Avoid use long labels. If the space is limited to display long label, truncated text with an ellipsis '...'.
  • Avoid using gradient color and 3-D bar charts.
  • Avoid using different colors for each data when there is only one category. (example: Africa is blue, America is green, Asia is orange.).
  • Do not show legend when only have one category.
