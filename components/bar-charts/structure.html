Bar charts are used to compare data across categories. They are especially effective when you have numerical data that can be categorized, for example, volume of scanned traffic in different continents.

  • Use qualitative colors to group items. For example, use blue for Ransomware, orange for web reputation.
  • Ensure the numerical axis starting at zero.
  • Avoid use long labels. Truncated your label and add an ellipsis (...) when the label is too long to fit into the provided space.
  • Avoid using gradient color and 3-D bar charts.
  • Avoid using different colors for data sets of the same category.
  • Do not show legends when there is only one category.
