Wrap an input and a label within span element to implement a checkbox under the dropdown menu items.

About checkbox style, please refer to Form Elements.

You can use checkboxes in a drop-down menu to indicate item selection. As a summary, you can display a count for the number of selected items or list them out. Separate the listed items with commas, and put ellipses (...) at the end to indicate omission if the list is too long to fit into one line.
