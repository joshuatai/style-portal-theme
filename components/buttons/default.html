Button Tag

The button tag is defined based on normalize.css.
Button Class

This button class can be used for , , and elements. For example, a Bootstrap button uses .btn as its button class.
Default Class

Define the general style by extending the button class. For example, a defult style button in Bootstrap is .btn-default.
Primary Class

Define the primary style by extending the button class. For example, a primary style button in Bootstrap is .btn-primary.
Emphasis Class

Define the emphasis style by extending the button class. For example, an emphasis style button in Bootstrap is .btn-danger.
Flat Class

Define the flat style by extending the button class.
Linking Class

Define the linking style by extending the button class. For example, a linking button in Bootstrap is .btn-link.
  • Default: Use the default style if the button does not belong to any of the category below.
  • Primary: Except for Log on, Log out, and linking buttons, use the primary style for the most important button on a UI page. For example, save and ok buttons.
  • Emphasis: Use the emphasis style for logon buttons.
  • Flat: Use the flat style for buttons that do not generally need additional attention, such as toolbar buttons or buttons in tables.
  • Linking: Use the linking style for hyperlinks or for action buttons when you want to simply the interface.
