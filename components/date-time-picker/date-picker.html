The date picker contains the date input field and the calender view. Users can either use the input field to type a date or by selecting a date in the calendar.

  • Clicking the input field to open the calender dropdown.
  • Click the arrows to view the previous and next days, months, or years
  • Click the month/year shows an overview of all months/years
  • The selected date is shown with a red background. The today day is indicated with red
  • If no default date is selected, show a placeholder help text to indicate the date format
  • Once select a date, the calendar closes and shows the selected date in the input field
  • Finsh typing the whole date, then display the selected date in the calender view
  • Support keyboard selection

For default date format is dd/mm/yyyy, other formats details for local dependencies, see Date & TimeUI guidelines.
